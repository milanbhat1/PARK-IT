# PARK-IT

A **Smart Parking Management Application** designed to simplify the parking experience through an intuitive and efficient interface.  
This project was created as part of a design and development initiative to address everyday parking challenges using modern UI/UX principles.

---

## Overview

**PARK-IT** helps users find and manage parking spots effortlessly.  
Through our Figma prototype, we designed a seamless flow from locating available spaces to booking and payment confirmation.
